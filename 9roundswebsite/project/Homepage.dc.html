The idea behind 9rounds

9rounds is a nine-station kickboxing circuit — you rotate through every station once, with no rest and no repeats. No two days run the same combination, so your body never settles into a routine. There are no class times to catch: the timer starts when you do. The format strips out the guesswork, so all you have to do is show up and work.
